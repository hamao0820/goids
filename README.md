# goids: boids algorithm CLI Command with Golang

![](goids.gif)

# 遊び方

```zsh
$ goids
```

gopher が boids flocking algorithm に従って動く様子が GUI で表示されます.  
まるで渡り鳥になった gopher くんを見て癒されましょう。

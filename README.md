# goids: boids algorithm CLI Command with Golang

![](goids.gif)

gopherくんがboids flocking algorithmに従って動くのを眺めるだけのアプリです.
まるで渡り鳥になったgopherくんを見て癒されましょう.

# 遊び方

```zsh
$ ./bin/goids
```

または, 以下のバイナリファイルをダブルクリックして起動してください.  
[./bin/mac/goids](./bin/goids)
